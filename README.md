# Kostenplan

Cost planning and management project.

## Setup for Claude Code

Claude Code is configured for this project. The following files enable integration:

- **`claude_info.json`** — Project metadata and configuration
- **`.claudecodeignore`** — Files/folders to exclude from Claude Code processing

## Getting Started

To work with Claude Code on this project:

1. Install dependencies: `npm install`
2. Start development: `npm run dev`
3. Run tests: `npm test`
